# 📊 Telecom X - Análisis de Churn de Clientes

## 📌 Descripción del Proyecto

Este proyecto forma parte del análisis de datos para **Telecom X**, empresa que enfrenta una alta tasa de evasión de clientes (Churn).

El objetivo principal es analizar los datos históricos de clientes para identificar patrones y factores asociados a la cancelación del servicio, utilizando Python y herramientas de análisis de datos.

Este análisis servirá como base para el desarrollo de modelos predictivos que permitan reducir la pérdida de clientes.

---

## 🎯 Objetivos

- Calcular la tasa general de Churn.
- Identificar variables relacionadas con la evasión.
- Analizar el comportamiento de los clientes según:
  - Género
  - Tipo de contrato
  - Método de pago
- Generar visualizaciones que faciliten la interpretación de los datos.
- Proponer recomendaciones estratégicas para reducir el Churn.

---

## 📈 Principales Hallazgos

- 🔹 Tasa general de Churn: **26.5%**
- 🔹 Diferencia mínima de Churn entre hombres y mujeres.
- 🔹 El contrato **Month-to-Month** presenta la mayor tasa de cancelación (1655 clientes).
- 🔹 El método de pago con mayor Churn es **Electronic Check** (1071 clientes).
